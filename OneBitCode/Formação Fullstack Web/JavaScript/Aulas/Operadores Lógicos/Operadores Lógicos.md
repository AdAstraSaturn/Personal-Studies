# Operadores Lógicos

-   AND (e)
    -   Só retorna verdadeiro **se ambas as partes forem verdadeiras**.
    -   operador `&&`
        -   `true && true === true`
        -   `true && false === false`
        -   `false && true === false`
        -   `false && false === false`
-   OR (ou)
    -   Só é falso **se ambas as partes forem falsas**.
    -   operador `||`
        -   `true || true === true`
        -   `true || false === true`
        -   `false || true === true`
        -   `false || false === false`
-   NOT (não)
